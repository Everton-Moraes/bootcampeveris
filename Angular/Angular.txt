Para instalar angular CLI

npm install -g @angular/cli

ng version // para saber a versão

ng new nome da aplicação // para criar um novo projeto

ng serve // para inicializar o servidor

npm istall bootstrap // para instalar o bootstrap


O angular é modular, ou seja, sempre terá um módulo
Módulo engloba toda a aplicação - limitador de contexto

componentes é tudo que será visualizado no módulo

One Way data binding - quando só exibe

two way data binding - quando exibe e edita
